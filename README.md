# Rohit Kumar — Portfolio

Personal portfolio website for **Rohit Kumar**, Senior Application Security Engineer.

🔗 **Live site:** https://r04i7.github.io/

## About

A self-contained, single-file static website (no build step required). Just open `index.html`
or serve the folder with any static web server.

- **Hero** with animated role typing and key stats
- **About** — bio, certifications, contact links
- **Experience** — full career timeline
- **Skills** — security testing, tools, standards, languages
- **Projects** — CVE research and security tooling
- **Contact** — links, contact form, and resume download

## Tech

Plain HTML, CSS, and vanilla JavaScript. No dependencies. Fonts via Google Fonts.

## Resume
